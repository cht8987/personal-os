# 移动端接入指南（可插拔适配器）

## 方案 A：iOS 快捷指令（默认启用，零服务器）

原理：快捷指令把文本写入同步目录的队列 `01-Inbox/.queue/`，
Hermes 每晚巡检（或手动 `rtk patrol`）时统一消化。

### 快捷指令搭建步骤（3 个动作）

1. **提问式交互**：添加 3 个「要求输入」动作，提示语依次为：
   - 「今天核心进展是什么？」
   - 「有产生新的待办或消费吗？」
   - 「还有什么随想要记录？」
2. **拼合文本**：用「文本」动作把三个回答按行拼接。
3. **存储文件**：保存为
   `PERSONAL OS/01-Inbox/.queue/q-当前日期时间.txt`
   （目标位置选择你同步盘中的本目录；Obsidian 库经 iCloud /
   Syncthing / NAS 同步均可）。

语音输入：把第 1 步换成「听写文本」即可，一句话全量入流，
由 AI 在巡检时自动拆分实体。

## 方案 B：Telegram Bot（预留，未启用）

启用条件：在 `os.config.yml` 把 `ingest.adapters.telegram.enabled` 设为 true，
配置 `TG_BOT_TOKEN` 环境变量，并部署一个把 Bot 消息转发为
`rtk ingest "<文本>"` 调用的常驻进程（建议运行在常开的 Mac/NAS 上）。
管线下游完全复用，无需任何其他改动。

## 方案 C：Mac/PC 命令行（内建）

```bash
rtk ingest "记得明天给服务器换证书，午饭花了38块"
```
